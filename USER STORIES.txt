US1: As a user, I want to see play options such as one or two player modes on game load 
	so that I can choose which mode to play. 
US2: As a user, I want the game to generate random puzzles for me to guess and reveal the solution 
	so that I can play single player
US3: As a user, I want to be able to drag and drop my pins for my guesses 
	so that the game is more engaging/easy to use.
US4: As a user, I want to receive correct feedback on my guesses 
	so that I know what I need to fix.
US5: As a user, I want to see the history of my guesses/results 
	so that I can remember what I’ve done.
US6: As a user, I want to add friends
	so that I can play two player with them. 
US7: As a user, I want to make my own puzzles 
	so that I can send it to my friend.
US8: As a user, I want to send feedback to my friend’s guesses 
	so that they know how much is right.
US9: As a user, I want to have the game validate my feedback in two-player 
	so that my friend is sure to get accurate results.
US10: As a user, I want simple navigation between main screens, with helpful/catchy graphics 
	so that the app is more user friendly.
